## Context方法

#### 保存状态
> 可以累加保存。
```
context.save();
```

#### 状态回滚
> 回滚到最近的一次状态。
```
context.restore();
```

#### 平移
> 平移坐标轴，会产生累加。
```
context.translate(x轴平移多少，y轴平移多少);
```

#### 旋转
> 旋转坐标轴，会产生累加。
```
context.rotate(弧度);
```

#### 缩放
> 缩放坐标轴，会产生累加。
```
context.scale(当前坐标系x轴缩放的倍数，当前坐标系y轴缩放的倍数);
```
